# Documentación de Testing Manual

## Aplicación testeada

* Nombre: SauceDemo
* Tipo de aplicación: E-commerce web
* URL: https://www.saucedemo.com/

---

Esta sección contiene toda la documentación de testing manual realizada sobre el proyecto SauceDemo.

## Estructura de la documentación

### 1. Test Plan

Documento que describe la estrategia de testing, alcance, objetivos, ambiente de pruebas y resumen de ejecución.

### 2. Historias de Usuario y Escenarios BDD

Requerimientos funcionales organizados por épicas, incluyendo criterios de aceptación en formato BDD.

### 3. Test Cases

Casos de prueba manuales diseñados y ejecutados para validar las funcionalidades del sistema.

### 4. Bug Reports

Defectos detectados durante el proceso de testing, incluyendo severidad, prioridad y pasos para reproducir.

### 5. Matriz de Trazabilidad

Relación entre épicas, historias de usuario, test cases y bugs detectados.
